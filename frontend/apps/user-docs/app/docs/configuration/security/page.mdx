## Quick Security Tips

<Callout variant="warning" title="Production Security">
Never expose rbee services directly to the internet without proper authentication and TLS.
</Callout>

**Loopback-only binding (default):**
```bash
# Services bind to localhost by default
queen-rbee --port 7833  # Binds to 127.0.0.1:7833
```

**Non-loopback requires API token:**
```bash
# For remote access, set API token
export LLORCH_API_TOKEN="your-secure-token"
queen-rbee --bind 0.0.0.0 --port 7833
```
