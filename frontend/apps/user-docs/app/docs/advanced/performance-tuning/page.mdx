## Quick Tips

**Resource limits (Linux):**
```bash
# Limit worker to 2 CPU cores and 4GB RAM
rbee-hive spawn-worker \
  --model llama-3-8b \
  --worker cuda \
  --device 0 \
  --cpu-limit "200%" \
  --memory-limit "4G"
```

**Monitor performance:**
```bash
# Watch telemetry stream
curl -N http://localhost:7833/v1/heartbeats/stream
```
