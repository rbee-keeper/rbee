## Known Issues (Non-blocking)

### Tailwind v4 `@source` Syntax
```
app/globals.css:19:2 parse
√ó Tailwind-specific syntax is disabled.
```

**Status:** Expected - Biome doesn't recognize Tailwind v4's `@source` directive yet  
**Impact:** None - This is a parser warning, doesn't affect functionality  
**Workaround:** Ignore or add to biome.json ignores if needed

### Generated Type Files
Some generated `.d.ts` files (like `worker-configuration.d.ts`) have `any` types.

**Status:** Expected - Generated by Wrangler  
**Impact:** None - These are external type definitions  
**Solution:** Already ignored in `biome.json`
